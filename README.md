# GoSeach

`Google` Search CLI base on [RobertChristopher/go-google](https://github.com/RobertChristopher/go-google)

Use ENV `https_proxy` for proxy

## Usage

```
$ go get -u  github.com/ryh/gosearch/cmd/gos
```
use `gos` to search
